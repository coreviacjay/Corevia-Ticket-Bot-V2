# Corevia Ticket Bot V2

Official ticket bot for Corevia Global.

## Features
- Category selection dropdown
- Private ticket channels
- Claim / Unclaim button
- Close button + auto-transcript to logs
- Clean embeds matching Corevia style

## Setup
1. Create bot → https://discord.com/developers
2. Invite bot with scopes: bot + applications.commands
3. Fill config.json
4. npm install
5. npm start
6. Run /setup in a public channel

Made from Corevia Global Network.
